# 📊 AI Job Market Insights Dashboard

An interactive Tableau dashboard project analyzing AI and technology job market trends using real-world job posting data.

This project focuses on salary analysis, hiring trends, industry demand, company hiring activity, and in-demand technical skills through engaging and colorful data visualizations.

---

# 🚀 Project Overview

The purpose of this project was to transform raw job posting data into meaningful business insights using Tableau and data storytelling techniques.

The dashboard explores:

✨ Highest-paying job roles by experience level  
✨ Industries with the most job opportunities  
✨ Most requested skillset combinations  
✨ Entry-level hiring trends by company size  

This project was completed as part of the **DataWorks Data Visualization Challenge**.

# 📌 Dashboard Questions & Analysis

## 💰 1. Highest Paying Jobs Based on Experience

- Calculated the average salary for each job title across different experience levels
- Identified the top 3 highest-paying job titles within each experience category
- Used comparative bar charts for clear visualization

### 🧠 Skills Demonstrated
- Aggregation
- Sorting & Ranking
- Salary Analysis
- Comparative Visualization

---

## 🏢 2. Industries with the Most Job Opportunities

- Identified the Top 5 industries with the highest number of job postings
- Found the top 3 companies with the most vacancies within those industries
- Applied calculated fields and ranking filters using Tableau functions

### 🧠 Skills Demonstrated
- Calculated Fields
- RANK_UNIQUE Function
- Filtering
- Industry Analysis

---

## 🧩 3. Most Requested Skillset Combination

- Analyzed the most frequently requested combinations of technical skills
- Used bubble charts to highlight high-demand skillsets
- Applied filters to focus on meaningful job posting frequencies

### 🧠 Skills Demonstrated
- Skill Frequency Analysis
- Bubble Charts
- Data Filtering
- Pattern Recognition

---

## 📅 4. Entry-Level Job Opportunities by Company Size

- Analyzed monthly hiring trends for full-time entry-level roles
- Compared hiring activity across:
  - 🟢 Small companies
  - 🟡 Medium companies
  - 🔵 Large companies
- Built a multi-line trend visualization with legends and color coding

### 🧠 Skills Demonstrated
- Time Series Analysis
- Trend Visualization
- Multi-Line Graphs
